# oommf-in-spack

Dedicated branch to periodically attempt to compile OOMMF using Spack with the `develop` branch. This tends to be unstable.

Support repository for getting the [OOMMF](https://math.nist.gov/oommf/) package into [Spack](http://spack.readthedocs.io).
